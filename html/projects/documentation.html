Install Python

In the command line, run:

sudo apt install -y python3

If prompted, enter your password and press Enter

Create a Program File

Navigate to the desired program location in a terminal

In the command line, run:

vim hello_world.py
Write a Program

Press i to begin typing in vim

Type or copy the following lines:

string="Hello World"
Print (string)

To save and close the file, type Esc followed by :wq

Run the Program

In the command line, run:

python3 hello_world.py

You should see the output Hello World in the terminal
